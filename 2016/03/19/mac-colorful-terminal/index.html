Mac Colorful Terminal

2016-03-19
  1. 前言
  2. 具体步骤
  3. Terminal设置
  4. Vim设置
  5. ls设置
  6. 参考

前言

今天装完Mysql之后发现之前终端的颜色不见了，本来那颜色看的也挺操蛋的。Google之后发现了solarized。来看看效果👇

具体步骤

  1. Terminal设置
  2. Vim设置
  3. ls设置

Terminal设置

  1. github下载solarized

         $ git clone git://github.com/altercation/solarized.git

  2. 导入terminal配置文件

    打开Terminal的偏好设置，导入下面的配置文件

         solarized/osx-terminal.app-colors-solarized/Solarized Dark ansi.terminal

Vim设置

Vim保持和Terminal相同的配色，设置如下👇

        $ cd solarized
        $ cd vim-colors-solarized/colors
        $ mkdir -p ~/.vim/colors
        $ cp solarized.vim ~/.vim/colors/

        $ vim ~/.vimrc
        syntax enable
        set background=dark
        colorscheme solarized

ls设置

  1. 安装Coreutils
         $ brew install coreutils
  2. 生成颜色配置文件
         $ gdircolors --print-database > ~/.dir_colors
  3. 编辑配置文件(.bash_profile)
    在.bash_profile文件中添加下面几行👇
         if brew list | grep coreutils > /dev/null ; then
         PATH="$(brew --prefix coreutils)/libexec/gnubin:$PATH"
         alias ls='ls -F --show-control-chars --color=auto'
         eval `gdircolors -b $HOME/.dir_colors`
     fi

p.s 照做就好了=。=
